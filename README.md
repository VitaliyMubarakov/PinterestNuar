# Pinterest Nuar
![Main](./img/VERY_BIG.png)
### Customizable dark theme for pinterest.com

## Features
- Uses the original color palette.
- Does not invert colors.
- Supports settings menu.

- The User can change: 
    1. Rounding of images
    2. Corners indentation
